# LearnCpp_Boost_EchoServer

В рамках изучения многопоточности и асинхронности в C++ был написан простой эхо сервер с использованием библиотекм Boost.Asio и системы сборки CMake. При запуске программы мы создаём объект io_context, который является нвашим event loop`ом для асинхрорнных операций, далее создаётся объект класса Server принимая в качестве аргументов io_context и порт, который будел слушать ссервер. Далее запускает наш event loop, в котором при подключении создаётся объект класса Session, контруктор которого требует в качестве аргумента сокет подключения. После запуска сесии, сервер считывает данные и возвращает их обратно.
